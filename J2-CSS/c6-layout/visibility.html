visibility: visible

visibility:visible; 使用屬性值 visibility:visible; 來顯示元素。

visibility: hidden

visibility:hidden; 使用屬性值 visibility:hidden; 來隱藏元素。指定 hidden 的時候，雖然元素是不可見的，但是它所佔用的空間仍然存在。

visibility: collapse

visibility:collapse 使用屬性值 visibility:collapse; 來隱藏元素。多數時候與 visibility:hidden; 相似，但是用於 table row/column 時類似 display:none;

display: none

display:none 使用display:none 隱藏元素。元素不可見，也不佔用空間。
